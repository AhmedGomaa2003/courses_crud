# Courses API

## 📌 Description
RESTful API built with Node.js and MongoDB to manage courses.

## 🚀 Features
- Create course
- Get all courses
- Get single course
- Update course
- Delete course
- Pagination

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB

## ▶️ How to Run
1. npm install
2. npm start

## 🔗 API Endpoints
GET /api/courses  
POST /api/courses  
GET /api/courses/:id  
PATCH /api/courses/:id  
DELETE /api/courses/:id
